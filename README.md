# Eyes - AI 공감형 스마트 복약 케어 시스템

## 프로젝트 소개
노인 및 시각장애인을 위한 AI 기반 스마트 복약 관리 시스템입니다.
라즈베리파이 5를 중심으로 카메라, LED, 로터리 다이얼 인코더를 활용합니다.

## 주요 기능
- 약 촬영 및 OCR 인식
- AI 기반 복약 정보 안내 (음성 출력)
- QR 코드 보안 인증
- 음성 인식 (STT) 기반 사용자 인터랙션
- 복약 스케줄 관리

## 기술 스택
- **Hardware**: Raspberry Pi 5, NeoPixel LED, 로터리 인코더
- **Language**: Python 3.11
- **AI**: Anthropic Claude API, OpenCV, OCR
- **DB**: SQLite
